# Code of Conduct

## Our Commitment
We are committed to providing a professional, respectful, and inclusive
environment for contributors and users.

This project is intended for defensive privacy, research, and authorized
security testing.

## Expected Behavior
Participants are expected to:
- Act professionally and respectfully
- Engage in constructive technical discussion
- Respect differing viewpoints and experiences
- Use the project responsibly and lawfully
- Follow applicable laws and ethical guidelines

## Unacceptable Behavior
Unacceptable behavior includes:
- Encouraging or promoting illegal activity
- Requests to add evasion, fraud, or bypass features
- Harassment, abuse, or discriminatory language
- Misrepresenting the projectâ€™s purpose or capabilities
- Attempting to repurpose the project for wrongdoing

## Scope
This Code of Conduct applies to:
- GitHub repositories
- Issues and pull requests
- Discussions and documentation
- Any related communication channels

## Enforcement
Project maintainers may:
- Remove content
- Reject contributions
- Restrict or revoke participation

Decisions are made to protect the integrity, legality, and professional
standing of the project.

## Reporting
If you observe violations of this Code of Conduct, report them to the
project maintainers privately.
